# Kalkulator
Pembuatan kalkulator dengan fungsi tambah, kurang, cancel, dan exit.
## Dev Environment
1. [Java OpenJDK11](https://adoptopenjdk.net/?variant=openjdk11&jvmVariant=hotspot)
2. [Gradle](https://gradle.org/install/)
## Build Instructions
```
./gradlew clean build
```
## Run Instructions
```
./gradlew run
```
## Run Test
```
./gradlew clean test
```
